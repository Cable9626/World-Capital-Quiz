# World Capital Quiz

A web application that quizzes users on world capitals.

## Features

- 🌍 Random country questions
- ✅ Answer validation
- 📈 Session-based scoring
- 🔄 Restart game after an incorrect answer
- 🗄️ PostgreSQL database hosted on Neon

## Built With

- Node.js
- Express
- EJS
- PostgreSQL
- express-session

## Getting Started

1. Clone the repository.
2. Run `npm install`.
3. Create a `.env` file with your PostgreSQL connection string.
4. Run `npm start`.
